Detailed Description

template<typename T>
class DS::LinkedListSingle< T >

Implements a mple linked-list Characteristics: Non-contiguous Space Complexity: O(n) Time Complexity:

  • Indexing: O(n)
  • Search: O(n)
  • Insert: O(1)
  • Delete: O(1)
  • Push/Pop: O(1) front O(n) back

Detailed Description

template<typename T>
class DS::LinkedListDouble< T >

Implements a double linked-list Characteristics: Non-contiguous Space Complexity: O(n) Time Complexity:

  • Indexing: O(n)
  • Search: O(n)
  • Insert: O(1)
  • Delete: O(1)
  • Push/Pop: O(1) front/back

Detailed Description

template<typename T>
class DS::StackArray< T >

Implements an array-based stack Characteristics: Space Complexity: O(n) Time Complexity:

  • Indexing: O(1)
  • Insert: O(1)
  • Delete: O(1)
  • Push/Pop: O(1)

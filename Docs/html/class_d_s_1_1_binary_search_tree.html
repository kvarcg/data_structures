Detailed Description

template<typename T>
class DS::BinarySearchTree< T >

Implements an ordered binary tree Characteristics: Non-contiguous Space Complexity: O(n) Time Complexity:

  • Search: O(h) or O(logn)
  • Insert: O(h) or O(logn)
  • Delete: O(h) or O(logn)
